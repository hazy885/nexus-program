Run the evidence-based ideation pipeline. Don't just list the first ideas that come to mind.

## 1. Frame as a Job-to-be-Done
Restate the goal as the *need/job*, not a solution ("users want a hole, not a drill"). State who the user is and what success looks like. This is the antidote to the #1 failure cause: building something with no real need. If the job is unclear, ask one sharp question.

## 2. Diverge — generate widely, defer judgment (quantity first)
Generate a broad set (aim for 8–15), deliberately mixing lenses so you don't anchor on the obvious:
- **First-principles** — break the problem to fundamentals, rebuild from function.
- **Combinatorial / analogical** — borrow mechanisms from adjacent domains/products.
- **SCAMPER** — Substitute, Combine, Adapt, Modify, Put-to-other-use, Eliminate, Reverse on the current thing.
Do NOT judge or filter yet. Include at least 2–3 deliberately non-obvious / contrarian options.
> Evidence note: individual generation beats live group brainstorming (production blocking). Generate first, converge second.

## 3. Converge — cluster then screen
- Group ideas into themes; drop duplicates.
- Screen each on **Desirability** (do users want it?), **Feasibility** (can we build/maintain it?), **Viability** (does it help the business?). Order matters — desirability first.
- Score the shortlist with **ICE** (Impact × Confidence × Ease, 1–10). Show the table. Flag that scores are estimates, not proof.

## 4. De-risk the top idea(s)
For the top 1–2: name the **single riskiest assumption** (usually desirability), and propose the **cheapest experiment that could disprove it** (interview, fake-door test, concierge MVP, prototype) — before any real build.

## 5. Decide + capture
- Recommend persevere / test / park, with one-line reasoning each. Two-way door → just run the cheap test; one-way door → flag for a deliberate decision.
- Capture the idea set, ICE scores, chosen bet, and next experiment so killed ideas aren't re-litigated.

Be honest: surface pros AND cons per shortlisted idea, and say where confidence is low. No hype.
